# ArabicPoetryManagementSystem



Poem.txt is the dataset for the poetry management system. You have to import these poems.



• Ignore footnotes, they are after the line containing "\_\_\_\_\_\_\_\_\_\_"

• "===========" means end of page

• Poem title is enclosed within \[ and ]. For example, \[قال قُرَيْطُ بنُ أُنَيْفٍ أحد بني العنبر]

• Verses are enclosed within ( and ). For example, ( لَوْ كُنْتِ مِنْ مَازِنٍ لَمْ تَسْتَبِحْ إِبْلِي ... تَنُو ) ( اللَّقِيطَةُ مِنْ ذُهْلِ بْنِ شَيْبَانَ )
